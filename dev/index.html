MaterialPointVisualizer.jlMPM model postprocessor

A High-performance Backend-agnostic Material Point Method Solver in Julia

View on GitHub 👀
🧰

Compatibility

Integrated with MaterialPointSolver.jl, it can be used directly to visualize calculation results; there is also a convenient API for general particle data.

🔍

Surface Reconstruction

To better carry out post-processing, we use splashsurf to reconstruct the surface from the MPM particle model, including multiple file sequences.

👷

VTK

Support exporting to .vtp files or animations for easy viewing or rendering in ParaView.

🖥️

GUI

We built a interactive GUI that supports millions of particles using Three.js, which can automatically return to the local browser for immediate viewing even on a remote headless server.
